# 创作工坊 - 小说创作助手

一站式小说创作辅助工具，内置科学计算器和多种 AI 生成功能。

## 功能

- **科学计算器** — 支持三角函数、对数、幂运算，完整键盘支持
- **角色起名** — 根据小说类型和性别智能生成角色名
- **角色设定生成** — 一键生成名字、性格、身世、能力、外貌
- **背景设定生成** — 生成世界观、力量体系、势力格局、地理环境
- **小说起名** — 按类型生成朗朗上口的书名
- **封面生成** — 自动生成风格化小说封面，可下载 PNG

## 技术栈

- 后端：Python Flask
- 数据库：SQLite（数据保存在 `data/app.db`，便于迁移）
- 前端：原生 HTML/CSS/JS + math.js
- AI：内置模板引擎 + 可选 Ollama 本地模型增强

## 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动服务
python app.py

# 3. 打开浏览器
# 访问 http://localhost:5000
```

## 可选：启用 AI 增强

安装 [Ollama](https://ollama.ai) 后运行：

```bash
# 安装轻量中文模型
ollama pull qwen2:0.5b

# Ollama 默认运行在 11434 端口，应用会自动检测
```

开启后，生成器界面中的「使用 AI 增强」开关即可使用。

## 数据迁移

所有数据保存在 `data/app.db` 单个 SQLite 文件中，复制即可迁移。
